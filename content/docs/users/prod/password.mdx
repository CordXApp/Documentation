---
title: Password Updates
description: Information about how to change your default password.
icon: CogIcon
---

When you are first given your CordX Account we will set it up using a default password
this password can be changed on initial login by following these steps:

-   Head to your dashboard: []()
-   Navigate to the `settings` tab
-   Scroll down the page to the `Update password` section
-   Set your password using the provided text box
-   Click the submit button to save your changes.
