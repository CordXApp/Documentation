## SFW/SFL Domains

<Callout title="NOTE" type="info">
    Wildcards can be used with any subdomain for example `wildcard.cordx.lol` however they will not be able to login to
    the website, using the website without a wildcard for example `cordx.lol` will allow you to be able to login.
</Callout>
