This is a simple api we use for generating memes from the
subreddit api into our discord bot. We decided to open it up
to the public, enjoy xD

---

## Available endpoints

-   Random Memes: `https://api.cordx.lol/v3/client/memes`
-   Dank Memes: `https://api.cordx.lol/v3/client/memes/dank`
-   Danker Memes: `https://api.cordx.lol/v3/client/memes/danker`
-   Prequel Memes: `https://api.cordx.lol/v3/client/memes/prequel`
-   Facebook Memes: `https://api.cordx.lol/v3/client/memes/facebook`
-   Wholesome Memes: `https://api.cordx.lol/v3/client/memes/wholesome`
-   Deepfried Memes: `https://api.cordx.lol/v3/client/memes/deepfried`
-   Surreal Memes: `https://api.cordx.lol/v3/client/memes/surreal`
-   Funny Memes: `https://api.cordx.lol/v3/client/memes/funny`
-   More Random: `https://api.cordx.lol/v3/client/memes/last`
-   Economy Memes: `https://api.cordx.lol/v3/client/memes/economy`
-   Cat Memes: `https://api.cordx.lol/v3/client/memes/cats`
-   Dog Memes: `https://api.cordx.lol/v3/client/memes/dogs`

## Example usage

```js title="Example Request"
const fetch = require('node-fetch');

await fetch('https://api.cordx.lol/v3/client/memes')
.then((res) res.json())
.then((meme) => {

    await console.log(`${meme.title} | ${meme.author} | ${meme.url}`)


}).catch((e) => console.log(e.stack));
```
